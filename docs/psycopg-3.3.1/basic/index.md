# Getting started with Psycopg 3

This section of the documentation will explain [how to install Psycopg](install.md#installation) and how to perform normal activities such as [querying
the database](usage.md#usage) or [loading data using COPY](copy.md#copy).

#### IMPORTANT
If you are familiar with psycopg2 please take a look at
[Differences from psycopg2](from_pg2.md#from-psycopg2) to see what is changed.
